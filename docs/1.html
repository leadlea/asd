レビュー解説（高校生にもわかる）

ASDと言語の特徴 — 凸凹（でこぼこ）を前提にする視点

ASDの「ことば」を総説的にわかりやすく整理。文法・語彙・発話・語用論・心の読み取りなど、複数の角度から“山と谷”の凸凹を見ていきます。

1. どんな論文？

自閉スペクトラム症（ASD）の「ことば」の特徴を、これまでの研究を整理してわかりやすくまとめた総説（レビュー）です。ASDの人たちの言語は一様ではなく、得意・苦手が“山と谷”のように凸凹しているのが大きなポイントです。

2. 3つの代表的な“ことばのプロファイル”

ASD-LN（Language Normal = 構造的言語はふつう）

文法や語のルール（音・形・語順など）は保たれている一方、相手や場面に合わせて伝えるなどの語用論（コミュ力の言語面）でつまずきやすいタイプ。

ASD-LI（Language Impaired = 構造的言語に弱さ）

音や語の形、文法など言語の“ルール面”で弱さが見られるタイプ。発達性言語障害（DLD）と似る場合もあるが、まったく同じではない点に注意。

MV（Minimally Verbal = 最小限の発話）

話す言葉がほとんどない／ごく少ないタイプ。学齢期〜成人の約3割がMVのままという報告もあり、家族・臨床ともに支援ニーズが高い領域。

3. よくある“つまずきどころ”

語用論（コミュ場面での言葉の使い方）

ジョーク・皮肉・遠回し表現・比喩（ことわざなど）が難しい、相手の気持ちや考え（心の理論: ToM）を言語でくみ取るのが難しい、といった傾向。ただし個人差は大きい。

構造的言語（音・語・文法）

研究によってはASDの約半数前後で何らかの弱さが報告。ただし、対象の偏りや課題の難易度により数字の解釈は慎重に。

4. 「話さない＝わからない」ではない

MVの人でも、映像と言葉の対応を見る研究などから、主語-動詞-目的語（SVO）といった文の基本ルールを内側では理解している可能性が示されています。つまり、言葉が出にくくても“中の文法”は育っていることがある、という示唆です。

5. 研究・支援でのカギ（なにを評価する？）

分けて評価する

  • 語彙・文法（構造）
  • 語用論（会話運用、参照・比喩の理解など）
  • 韻律（話し方の抑揚・間・話速）

言語外の認知もみる

  • ToM（相手の心の推測）
  • 実行機能（EF: 注意・柔軟性・ワーキングメモリ）
  • できれば非言語的なやり方で評価
MVの人にも使える、積極的な返答を求めない理解度テスト（視線・選択・マッチング等）の開発・活用が重要。

6. 論文の結論（高校生向けにひとことで）

ASDの「ことば」は一人ひとりで凸凹が違う。だから、文法・語彙・発話・コミュ力・心の読み取りなどいろんな角度から見て、その人に合った学び方・伝え方を考えよう——というメッセージです。研究でも、細かなプロファイル分けと評価ツールの整備がこれからの課題です。

まとめ：ASDの「ことば」は多様。構造・語用論・韻律・非言語を分けて評価し、個別化した学び・支援へ。
