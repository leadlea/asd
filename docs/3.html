レビュー解説（高校生にもわかる）

ASDと言語・音声の特徴 — もう1本の総説をやさしく解説

ASD（自閉スペクトラム症）における言語・音声プロフィールの多様性を扱った総説の要点を、実践で使える目線で整理します。DSM-5/ICD-11の枠組み、言語サブシステム（意味・語用・形態統語・音韻/韻律・流暢性/運動）を横断し、評価と介入に直結するポイントをまとめました。

1. この論文のねらい

ASDの人たちに見られるコミュニケーションの困難さの組み合わせに基づいて、言語・音声のプロフィールを分類し、臨床・教育の鑑別や介入選択に役立つカテゴリー別アプローチを提案しています。個々の強みと弱みが連続体として広がる点が大前提です。

2. 診断枠組み（DSM-5 / ICD-11）

DSM-5の視点

  • ASDの中核に「社会的コミュニケーションの困難」。
  • 言語障害は併存・指定子で切り分ける。

ICD-11の視点

  • 知的発達障害の有無 × 機能的言語の有無でASDをタイプ分け。
  • 単なる「話せる/話せない」でなく、機能的・社会的な使用を重視。
診断ラベルは出発点。評価では言語の構造と語用論、韻律、非言語認知まで分けて見るのがコツ。

3. 言語プロファイル（4タイプ）

タイプ1：語用論中心の困難（他の言語障害なし）

言語能力は良好でも、社会的場面での使い分け（比喩・皮肉・談話の結束など）に課題。いわゆる“ペダンティック”な話し方や字義通り解釈が目立つことも。

タイプ2：ASD＋言語障害（DLD/SSD/言語失行など）

形態統語・音韻などルール面の弱さを併存。鑑別上、ASD起因の語用論課題とDLD由来の言語課題を分けて評価する。

タイプ3：知的障害を伴うASD

言語と一般知能の広範な遅れが関与。予後・教育支援では実用的コミュニケーションの確立が鍵。

タイプ4：二次的な言語遅れ

社会的交流・言語練習の機会不足等により、二次的に言語が遅れるタイプ。環境調整と言語入力の質・量が重要。

4. サブシステム別の所見

意味論

  • 抽象語の扱い・多義語・カテゴリー化でばらつき。
  • 一部では卓越した言語能力も報告（個人差が大きい）。

語用論

  • 比喩・皮肉・談話構成、共同注意の弱さと関連。
  • エコラリアやペダンティック・スピーチが機能的役割を持つ場合も。

形態統語

  • 時制標識・代名詞・クリティック等に課題が出るサブグループ。
  • 表面上は統語だが、語用論・韻律との接点由来の誤りも。

音韻 / 韻律

  • SSD併存の報告、語音の多様性制限、非典型韻律（単調・抑揚の不一致・話速の違いなど）。

流暢性 / 口腔運動

  • 吃音・クラッタリング様の非流暢、口腔運動の困難が一部で言語表出と関連。

関連する認知

  • ToM（心の理論）/ EF（実行機能）が言語使用（特に語用論）と結びつく。

5. 予測因子・早期介入

  • 乳幼児期の共同注意・模倣・遊びは後の言語・社会性の予測因子。
  • 親の言語入力の質・量が神経発達指標や到達に関連。
  • MV（最小限発話）でも、SVOなど基本語順の理解が内在する可能性—非言語的な理解課題の活用が有効。

6. 実践への示唆

  • プロファイル別に介入を設計（語用論中心／DLD併存／知的障害併存／二次遅れ）。
  • 単語テストだけでなく、語用論・韻律・談話＋ToM/EFまで評価。
  • 口腔運動・発話計画の課題が疑われるときは、運動面と並行した支援を。
  • 比喩・皮肉・参照の教示、ロールプレイ、視覚支援、一貫した環境調整を組み合わせる。

7. ひとことで要点（高校生向け）

ASDの言葉と話し方はすごく多様。文法・語彙・韻律・話し方・コミュ力・心の読み取りを分けて見て、その人のプロファイルに合う学び方・伝え方を選ぼう。診断名だけでは足りないので、タイプ×評価軸で支援をデザインするのがコツ！

まとめ：ASD×言語は“多様性”が前提。プロファイル別に、意味・語用・形態統語・音韻/韻律・流暢性・ToM/EFを横断して評価・支援する。
