# ADR-0001: v3draft への宗田さん指摘・表現修正の反映方針

- 状態: Accepted
- 日付: 2026-06-21
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（原文/正本）, `paper1_ja_v3draft.tex`（討議ドラフト）, `docs/homework/docx_comments_inventory.md`, `docs/minutes/mtg_20260622_expression_review.md`, `docs/translation/glossary_en.md`

## コンテキスト

2026-06-22 の山下先生MTGに向け、宗田さんの構成案・指摘および表現精査の結果を反映した討議用PDF（`paper1_ja_v3draft.pdf`）を作成している。
当初は「レビュー待ち・要相談の決定を確定版に先回りで焼き込まない」方針で一部を保留していた（品質フィルタの言い換え等）。

本MTGでは **`paper1_ja.pdf`（原文）を議論のベース**とすることが確定したため、討議ドラフト側（v3draft）には反映を進めてよいとの判断に至った。
ただし、誤反映・content喪失・査読耐性低下を防ぐため、「テキストとして安全かつ根拠が文書化済みの項目のみ即反映、それ以外は理由付きで保留」とする。

## 決定

1. 原文ベースは `paper1_ja.tex` / `paper1_ja.pdf` を維持（無改変）。
2. 反映先は `paper1_ja_v3draft.tex` のみ。再現スクリプトを git に残す。
3. 反映可否を下表で分類し、保留項目は理由を明示して将来対応とする。

## 帰結

- 討議ドラフトは安全な確定分のみ反映され、未確定・要相談はPDF上も原文（paper1_ja.pdf）で対比可能。
- 保留項目は本ADRで追跡され、MTG/英訳/図表再生成の各フェーズで解消する。
- 再現性: `scripts/build/restructure_v3draft.py`（構成）, `scripts/build/apply_soda_v3draft.py`（用語・表現）, `scripts/build/unify_big5.py`（Big5）に記録。
