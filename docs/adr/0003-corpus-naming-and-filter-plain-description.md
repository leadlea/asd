# ADR-0003: 「home2」「分析適格性フィルタ」造語の廃止とCONSORT図の削除

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（2.1節 データ）, ADR-0001（用語言い換えの初期方針）, `docs/minutes/reconcile_soda_yamashita_20260622.md`

## コンテキスト

最新PDF（2.1節）のレビューで、以下が判明した。

1. **「home2サブセット」は正式名称ではない**。福原が便宜的に命名した内部ラベルであり、CEJCに「home2」という公式サブセットは存在しない。論文に残すと、読者が公式区分と誤認する恐れがある。
2. **「分析適格性フィルタ（analytic eligibility filter）」も同様に造語**。ADR-0001で「品質フィルタ」の価値判断トーン回避のため導入した言い換えだが、英語の正式術語が存在するかのような体裁になっていた。
3. 2.1節のCONSORT形式フローチャート（図1）は、選定数値を図に依存させていた。

## 決定

1. **「home2」表記を全廃**し、CEJCの実条件「収録場所が自宅かつ参加話者数が2名の会話」と淡々と記述する。
2. **「分析適格性フィルタ / analytic eligibility filter / 内部識別子HQ1」を全廃**し、節見出しを「分析対象レコードの選定基準」に変更。本研究が独自に行った選定過程と数値のみを事実として記述する。
3. **CONSORT図（図1, `fig_consort_flowchart.png`）を削除**。図に含まれていた選定数値（577→140→378→120、除外437/258）を本文に文章として畳み込み、情報を保全する。
