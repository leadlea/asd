# ADR-0004: 2.1データ節を外形へ縫合し、詳細をサプリメンタル（付録）へ移送

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（2.1節 / 付録）, ADR-0003

## コンテキスト

2.1データ節が4つのsubsubsection（コーパスと前処理／選定基準／メタ情報の紐付け／話者の重複と性別内訳）に細かく分割され、本文の流れを分断していた。
方法論の本文としては要点（アウトライン）で足り、再現に必要な詳細は査読者がいつでも参照できればよい。

## 決定

1. **2.1.2〜2.1.4のsubsubsection見出しを廃止**し、要点のみを2.1の連続した本文に縫合（段落分割しない）。2.1.1見出しも外し、2.1データ節を一続きの記述にする。
2. 捨てるのではなく、**現在の粒度の詳細を付録（補足）へ移送**。査読者が随時参照できるよう保全する。
