# ADR-0008: 2.2 特徴量導入の表現軽量化とtopic_drift早期記載の削除

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（2.2節）, ADR-0006/0007

## コンテキスト

2.2冒頭の特徴量導入に2つの問題があった。

1. 「会話分析・相互行為論の知見を踏まえて本研究で新たに定義した新規提案の特徴量」という表現は、概要段階で根拠論文を伴わずに「知見」を断言するため、査読者に「その知見を示す論文は？」と問われやすい。
2. 「IX_topic_drift_mean は IX_lex_overlap_mean との完全共線性（r=−1.00）のため除外し統制変数として扱った」という記述が、変数の定義前（概要段階）に唐突に置かれており、購読者には意味が伝わらない。

## 決定

1. 概要の表現を軽量化: 「会話分析・相互行為論の知見を踏まえて本研究で新たに定義した」→「会話の相互行為的側面に着目して本研究で新たに定義した」。さらっと流れる表現にし、未裏付けの断言を回避。
2. 概要のtopic_drift共線性・除外の一文を**削除**。情報は変数定義後の「統制変数の除外と欠損値処理」（2.2内、現行294行付近）に既に記載されており欠落しない。
3. 山下6/22削除済み文に関する陳腐化コメントも併せて除去。

## 影響・整合

- 「新規提案の特徴量」の詳細節（2.2.x Novel Features）は、CA系の理論的根拠を `\cite{sacks1974, schegloff1977, kita2007}` 等で裏付けて記述しており、そちらは維持（断言ではなく出典付き）。
- 限界節（現行1589行付近）「IX_topic_drift_meanを統制変数として除外した（2.2節参照）」は、除外記載が2.2内の統制変数節に残るため参照は有効。
- topic_drift関連の他記述（相関構造3.2、記述統計3.1等）に影響なし。
