# ADR-0009: Novel特徴量の分類を「変数単位」と明示、PG_pause_variabilityの位置づけ整理

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（1章Intro(c), 2.2.2 Novel節）, ADR-0008

## コンテキスト

2.2.2 Novel節の導入に2つの問題。

1. 「会話分析（Conversation Analysis）および相互行為論の知見を踏まえて」が、根拠refの前に「知見」を断言し、査読者に出典を問われやすい。
2. 「相互行為構造系5変数（IX）と応答型系3変数（RESP），および沈黙変動性1変数（PG_pause_variability）から構成される」——新規カテゴリ2個（IX/RESP）の後に、Classicalであるはずのタイミング系（PG_）から1変数だけNovelに現れ、読者が「カテゴリが新規か／変数が新規か／同一カテゴリに既存と新規が混在するのか／本当に新規か」を判別できない。

### 原因（事実整理）

Classical／Novelは**カテゴリ単位ではなく変数単位**の区別である。タイミング系（PG）カテゴリは既存8変数（Classical）に加え、本研究で定義した沈黙長の変動係数（PG_pause_variability）1変数（Novel）を含む。すなわち同一カテゴリに既存・新規が混在する。

## 決定

1. 表現緩和: 「…知見を踏まえて」→「…で着目されてきた会話現象をもとに」。断言を避け、直後のref（schegloff1977/sacks1974等）で具体現象を裏付ける構成にする。
2. 分類粒度の明示: 「Classical／Novelの区別は特徴量カテゴリ単位ではなく個々の変数単位で行う」と本文に明記。IX・RESPは全変数が新規、PG_pause_variabilityは既存PGカテゴリに本研究で追加した1変数、と位置づける。
3. PG_pause_variabilityの新規性は「新規指標」と強く断言せず、「既存PGに加えた変数」「新たに定義」と控えめに表現（CVは単純統計量のため過度な新規性主張を避ける）。
4. 整合: 同一の構造的混乱がIntro(c)にも存在したため、Introも「既存のタイミング系（PG）に本研究で加えた沈黙変動性1変数」と修正し、知見表現も同様に緩和。

## 検討した別案（不採用）

- 別案B: PG_pause_variabilityをNovelリストから外し、Classical PGの拡張として別記。→ Novel総数9の数え方（表のClass列）と齟齬が出るため不採用。採用案（変数単位の明示）が表とも整合し最小変更。
