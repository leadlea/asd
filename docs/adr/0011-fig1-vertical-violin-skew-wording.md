# ADR-0011: 図1（縦型バイオリン）と本文の歪み表現の整合

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（3.1 記述統計, Discussion 記述的性質）, ADR-0010

## コンテキスト

図1（`fig_feature_distribution.png`）は**縦型バイオリンプロット**（縦軸=値、高値ほど上）であるのに、本文は分布の歪みを「右裾の長い分布」「右に偏った分布」と**左右前提**で記述しており、縦型の図と視覚的に矛盾していた（宗田さん由来の指摘系）。

## 決定

図ではなく本文を修正（図の再生成は影響大のため回避）。向きに依存しない統計的表現に統一する。縦型図では裾は高値側＝上方向に伸びるため、「高値側に裾を引く」「正の歪度」で表現する。

## 反映（paper1_ja.tex）

| 箇所 | 旧 | 新 |
|------|----|----|
| 3.1 PG | 概ね右裾の長い分布 | 概ね高値側に長く裾を引く分布（正の歪度） |
| 3.1 FILL | 右に偏った分布 | 高値側に裾を引く分布（正の歪度） |
| 3.1 OIR | 分布が強く右に偏っている | 分布は低値に集中し，強い正の歪度を示す |
| Discussion OIR | 分布が強く右に偏っている | 分布は低値に集中し強い正の歪度を示す |
