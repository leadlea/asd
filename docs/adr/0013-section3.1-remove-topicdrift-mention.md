# ADR-0013: 3.1から除外変数 IX_topic_drift_mean への言及を削除

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（3.1）, ADR-0012

## コンテキスト

ADR-0012でIXを5変数に同期したが、3.1に除外変数 `IX_topic_drift_mean（M=0.958）` を引用する一文（lex_overlap_meanとの補数・鏡像関係の説明）が残っていた。記述統計表には当該行がなく、5変数の記述と不整合だった。

## 決定

当該一文「IX_lex_overlap_mean（M=0.042）とIX_topic_drift_mean（M=0.958）は定義上の補数関係（1−Jaccard）にあり，分布は鏡像的である．」を**削除**（福原指示）。

## 検証

- 3.1内に `topic_drift` 言及の残存なし。
- uplatex 2パス＋dvipdfmx: 正常、47ページ、未解決の参照・引用なし。
- 補数関係・除外の根拠は2.2（統制変数の除外）および考察に記載が残るため、情報欠落の問題はない。

## 次の1手

最新PDFで3.1の整合を確認。次の個別修正対象の指示を待つ。
