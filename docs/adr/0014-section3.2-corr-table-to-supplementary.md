# ADR-0014: 3.2の相関数値表をサプリへ移送（図と重複）＋宗田さんColumn key削除指示の確認

- 状態: Accepted
- 日付: 2026-06-22
- 決定者: 福原玄（NCNP共同研究・筆頭）
- 関連: `paper1_ja.tex`（3.2 / 付録）, `reports/paper_figs_v2/tab_corr_matrix.tex`, `paper1_ja_st.docx`（宗田校閲）, ADR-0004/0012

## コンテキスト

3.2は (a) 19×19 Pearson相関の**数値表**、(b) カテゴリ別ブロック構造の**ヒートマップ図**、(c) 本文、の3点で同一データを提示しており重複が強い。
また「表の下の変数（数字）」＝相関表の Column key（(1)〜(19)の変数対応）について、宗田さんWordの指示を再確認する必要があった。

## 宗田さん指摘の再確認（事実）

`paper1_ja_st.docx` の校閲記録（著者=宗田さん, 2026-05-01）に、相関表（ヒートマップ）下の
`Column key: (1) PG_speech_ratio, … (19) RESP_YO_ENTROPY` を**削除（取り消し線）**するtrack-changeを確認。再追加（取り消し）は無し。→「削除指示」が正。

## 決定

1. 重複解消: インパクトの低い**19×19数値表（tab_corr_matrix）を付録\ref{sec:supp_corr}へ移送**。本文3.2はヒートマップ図＋本文を維持。
2. Column key の扱い: 宗田さんは本文（図）下のkeyを削除指示。本文からは数値表ごと消えるため、その意図（本文のkey clutter除去）は満たされる。一方、番号列(1)-(19)の数値表は付録でkeyが無いと解釈不能なため、**keyは数値表とともに付録にのみ残す**。

## 反映（paper1_ja.tex）

| 箇所 | 変更 |
|------|------|
| 3.2 冒頭 | 「表…と図…」→「図…にヒートマップを示す。数値行列(19×19)は付録\ref{sec:supp_corr}に掲載」。本文中のtable環境を削除 |
| 付録（新設） | `\section{特徴量間相関行列（補足）}`（`sec:supp_corr`）に数値表（Column key付き）を移送 |

## 検証

- `tab:corr_matrix` 参照は3.2冒頭の1件のみ→付録へ移送後も整合（label保持）。
- uplatex 2パス＋dvipdfmx: 正常、47ページ、未解決の参照・引用なし。

## 申し送り（要確認）

- 宗田さんはColumn key自体の削除を指示。本対応では「本文から除去（数値表ごと付録へ）＋付録の数値表解釈に必要なためkeyは付録に保持」とした。もし宗田さんの意図が「keyを完全削除（番号方式をやめ変数名表記へ）」であれば、数値表を変数名ヘッダで再生成する案に切り替え可能（要指示）。
