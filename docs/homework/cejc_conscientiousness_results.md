# Results / Discussion（CEJC自然対話：相互行為特徴 → LLM仮想教師C）

## Results
本研究では、自然対話における相互行為特徴が、LLM を仮想教師として得た人格質問紙推定値とどの程度整合するかを検証した。データは CEJC から「自宅・話者数2」の対話を抽出し、観測品質を担保するため HQ1 条件（`n_pairs_total≥80`、`FILL_text_len≥2000`、`IX_n_pairs_after_question≥10`）を満たす conversation×speaker を分析対象とした（N=120）。

仮想教師（目的変数）は IPIP-NEO-120 の C（Conscientiousness）24 項目を、Bedrock 上の LLM（Claude Sonnet 4）に strict 5 件法で回答させ、平均得点として算出した（`Y_C`）。説明変数には会話量（発話量・ペア数など）を含めず、相互行為の型に関する特徴量のみ（PG/FILL/IX/RESP から 18 変数）を用いた。Ridge 回帰を 5-fold cross-validation で評価した結果、予測値と仮想教師の相関は **r=0.335**、決定係数は **R²=0.111**であった。

さらに、目的変数をランダム置換した null 分布に対する置換検定（5000 回）により、同等以上の絶対相関が得られる確率は **p=0.0236**となり、観測された性能は偶然により説明されにくいことが示された（N=120, Xdim=18）。

係数の安定性について、ブートストラップ（500回、各回で Ridge を再学習）により上位特徴の再現性を評価した。トップ10に入る頻度（top-k 出現率）と係数符号の一致率（sign agreement）に基づくと、主要な寄与特徴は以下であり、いずれも高い再現性を示した（例：topk_rate≳0.56, sign_agree_rate≳0.88）。
- **フィラー出現（FILL_has_any）**：topk_rate=0.838, sign_agree=0.968
- **質問直後の修復開始（IX_oirmarker_after_question_rate）**：0.824, 0.984
- **発話比率（PG_speech_ratio）**：0.804, 0.932
- **応答遅れ（PG_resp_gap_mean）**：0.746, 0.986（負の係数）
- **NE直後相槌（RESP_NE_AIZUCHI_RATE）**：0.626, 0.914
- **YES/NO応答（IX_yesno_rate）**：0.568, 0.896
- **間（PG_pause_p50）**：0.564, 0.886（負の係数）

## Discussion
本結果は、自然対話における相互行為特徴が、LLM による人格質問紙推定（仮想教師）の少なくとも一部（C）と統計的に整合する可能性を示唆する。重要特徴の内容は、(i) 質問場面での修復（聞き返し）や (ii) 応答タイミングの滑らかさ（gap/pause）、(iii) 相槌や YES/NO といった応答型、(iv) フィラー出現といった、会話を成立させる運用機構に対応している。特に会話量を説明変数から外した上で置換検定が有意であった点は、単なる発話量や多弁さではなく、相互行為の「型」そのものが C 教師の変動と関係している可能性を支持する。

一方で、語彙重なりに基づく話題逸脱（topic_drift/lex_overlap）指標は、例文監査により短い相槌やコーパス付与タグの影響を強く受けることが確認された。タグ除去・短発話除外・相槌除外を施した driftv2 では性能と有意性が低下し（r=0.242, p=0.165）、少なくとも現行定義の drift は「話題逸脱」として解釈可能な信号と人工物が混在していた可能性が高い。したがって本稿では drift を主張軸から外し、修復・タイミング・応答型・フィラーに焦点を当てる。

本研究の新規性は、(1) 自然対話からの相互行為計測（PG/FILL/IX/RESP）と、(2) LLM を仮想教師として用いた人格質問紙推定を、(3) 監査可能性（例文監査→指標改訂→主張範囲の調整）を含む再現パイプラインとして統合した点にある。今後は、教師の頑健性（複数モデル ensemble）、相互行為特徴の拡張（質問—応答適合、修復成功/コスト）、および外的妥当性（別条件・別コーパス）を通じて、会話運用機構と仮想教師推定の対応関係を精緻化する。
