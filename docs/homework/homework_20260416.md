### 8. Dose-Response実験の最終結果（4教師ensemble確定）
- FILL ×0 / ×3 の2条件 × 4教師（Sonnet4, Qwen3, DeepSeek, GPT-OSS）× Cのみ = 8回の採点を完了
- **結果**: フィラー操作によるCスコアの有意な変化は検出されなかった
  - 4教師ensemble: ×0 mean C = 2.227, ×3 mean C = 2.215, ΔC = -0.012
  - 教師別: Sonnet4 ΔC=-0.003, Qwen3 ΔC=-0.023, DeepSeek ΔC=-0.023, GPT-OSS ΔC=0.000
  - 話者レベル: ↑51 / ↓63 / →6 (N=120)
- **解釈**: フィラー率はCの「指標」（相関的関連因子）だが「原因」（因果的決定因子）ではない
  - LLMはフィラーの数ではなく、会話全体の相互行為パターンを総合的に評価している可能性
  - Transformerのself-attention機構が個別フィラートークンより文脈構造に注目
