## 0. 環境・運用の前提（最初に読む）

- **Python実行は必ず `.venv/bin/python`**。`source env_asd.sh` だけでは環境が立たない。
- **LaTeXビルド**: `uplatex -interaction=nonstopmode <tex>` を2回 → `dvipdfmx <dvi>`。日本語はuplatex前提。
- **.gitignore対象**: PDF・補助ファイル(.aux/.dvi/.log/.out)・`artifacts/`・`reports/` 配下のデータ。
  - ただし `reports/paper_figs_v2/*.png` `*.tex` は追跡済みのものは `git add -f` で明示ステージする運用。
  - 再現はスクリプト＋texで担保する方針（データ本体はコミットしない）。
- **コミットは明示的に指示があったときのみ**。特定ファイルを名指しでadd（`git add .` は使わない）。
- 図表生成・分析スクリプトは `scripts/` 配下。論文図表は `reports/paper_figs_v2/`。
