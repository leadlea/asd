## 1. Methods 1節（2.1 データ）

### 現行ドラフトの骨子

2. **品質フィルタ（HQ1, 本研究独自）**: 3条件すべて満たすレコードを採用 — 発話ペア数≥80／テキスト≥2,000字／質問直後ペア数≥10。閾値は特徴量推定の安定性のため。

4. **話者の重複と性別内訳**: 74名のユニーク話者。25名が2件以上に参加し、重複話者由来レコードは71件（59.2%）。性別はユニーク話者で女38/男36、レコード単位で女66/男54。subject-wise split で情報リークを統制。
