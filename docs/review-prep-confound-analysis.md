# 査読対応準備: 交絡変数の統制（年齢・性別）

更新日: 2026-04-27
目的: 「年齢や性別の効果を統制したのか？」という査読質問に即座に回答できる資料

---

## 1. 分析設計

### 比較した 2 モデル

| モデル | 説明変数 | 変数数 |
|--------|---------|--------|
| Model A（特徴量のみ） | 19 相互行為特徴量 | 19 |
| Model B（交絡統制） | 19 特徴量 + gender + age | 21 |

### 手法

- Ridge 回帰（α=100）
- 5-fold CV（subject-wise split）
- Permutation test（n=5,000）
- 評価指標: Pearson r（予測値 vs 実測値）

### 交絡変数の定義

- `confound_gender`: M=0, F=1（ダミー変数）
- `confound_age`: 年齢帯の中央値（age_mid）

スクリプト: `scripts/analysis/confound_analysis.py`

## 4. C に関する結論（論文の主張を支える根拠）

1. **C は交絡統制後も頑健**: 3/4 教師で有意性が維持され、Δr はすべて正方向（+0.02〜+0.05）
2. **年齢・性別は C の予測を妨げない**: 交絡変数を追加しても r が下がらない → 特徴量が捉えている C の信号は年齢・性別とは独立
3. **DeepSeek-V3 は交絡統制の有無にかかわらず n.s.**: teacher 間一致度の低さ（C でも他 3 教師より低い）と整合的

## 5. 想定される査読質問と回答案

### Q1: 「年齢や性別の効果を統制したのか？」

**回答**: 19 特徴量のみのモデル（Model A）と、19 特徴量 + 性別 + 年齢の 21 変数モデル（Model B）を比較した。C（誠実性）は 3/4 教師で Model B でも有意性が維持され、Δr は +0.02〜+0.05 と正方向であった。これは、相互行為特徴量が捉えている C の信号が年齢・性別とは独立であることを示唆する。

### Q2: 「年齢・性別を入れると r が上がるのは、交絡ではなく情報追加では？」

**回答**: その通り。Δr が正方向であることは、年齢・性別が C の予測に追加的な情報を提供していることを意味する。重要なのは、特徴量のみのモデルでも有意であり、交絡変数を追加しても有意性が失われないことである。つまり、特徴量の予測力は年齢・性別の代理ではない。

### Q3: 「O（開放性）は年齢・性別を入れないと有意にならないのでは？」

**回答**: Sonnet 4 と Qwen3-235B では、特徴量のみでは n.s. だが交絡変数追加で有意になった。これは O の推定において年齢・性別が重要な予測因子であることを示唆する。O の結果は探索的として位置づけ、Discussion で年齢効果の可能性に言及する。

### Q4: 「N=120 で 21 変数は多すぎないか？」

**回答**: Ridge 回帰（α=100）は L2 正則化により多重共線性と過学習を抑制する。また、5-fold CV + permutation test（n=5,000）により、偶然の有意性を排除している。ただし、N=120 での 21 変数は自由度の観点で限界に近く、これ以上の共変量追加は推奨されない。これが年齢・性別のみを共変量とした理由の一つでもある。
