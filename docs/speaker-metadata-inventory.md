## 3. 年齢・性別だけにした理由（査読対応用）

### 統計的な理由

- N=120（CEJC）で Ridge 回帰に 19 特徴量を入れている。共変量を増やすと自由度が不足し、過学習リスクが上がる
- 年齢・性別は先行研究で Big Five との関連が確立されており、最低限統制すべき変数として選定

### データ品質の理由

- `birthplace` は 21 カテゴリに分散（最大でも東京都 37 件）。ダミー変数化すると次元爆発する
- `occupation` は 11 カテゴリだが「就学前」「小学生」等が混在し、年齢と強く交絡する
- `relationship` は欠損あり（4件）かつ「本人」が 58/120 で偏りが大きい。会話の相手との関係性は pairs 単位で定義すべきだが、現状は話者単位のラベルしかない

### CSJ 固有の理由

- `residence_years` は自由記述的（95 ユニーク値）で、そのままでは変数化が困難
- `education` は 4 カテゴリで使いやすいが、CSJ は追試データ（N=36）であり、CEJC と統一的に扱うには CEJC 側に学歴がない

## 4. 今後の可能性（Future work に書ける）

- **出身地 / 方言圏**: フィラーや終助詞の使用パターンは方言の影響を受ける。`birthplace` や `residence_years_lang_period`（言語形成期の居住歴）を方言圏にマッピングすれば、方言効果の統制が可能
- **関係性（親密度）**: `relationship` を「家族 / 友人 / その他」に集約すれば、親密度が相互行為特徴量に与える影響を検討できる
- **学歴**: CSJ には `education` があるため、CSJ 側で学歴効果の探索的分析が可能
- **両親の出身地**: CSJ の `father_birth_place` / `mother_birth_place` は、家庭内言語環境の代理変数として使える可能性がある
